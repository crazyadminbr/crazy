Основные команды (доступны всем)

/start

Описание: Запускает бота в личных сообщениях и отправляет приветственное сообщение с описанием возможностей.

Как работает:

  • Отправляет случайный стикер из массива stickers.
  • Выводит текст о функциях бота (роли, муты, приветствия).
  • Показывает главное меню с кнопками: "Выбрать Беседу", "Сетка Бесед", "FAQ по использованию".

Где работает: Только в личке.

/help

Описание: Показывает список доступных команд в зависимости от контекста и прав пользователя.

Как работает:

  • В личке: выводит базовые команды (/start, /help, /setnick).
  • В чате: проверяет права бота и пользователя (админ/владелец), выводит команды, доступные по роли (см. ниже).

Где работает: В личке и группах.

/setnick [имя]

+имя [имя]

Описание: Устанавливает пользователю ник в чате (до 20 символов).

Как работает: Сохраняет ник в статистике. Используется в сообщениях вместо имени..

Пример: /setnick Котик → "Ваш ник в чате установлен: Котик".

Где работает: Только в группах.

Команды модерации (требуют роли администратора)

/roles

роли

Описание: Показывает список администраторов чата с их ролями.

Как работает: Запрашивает данные из таблиц Роли Пользователей и Роли, выводит список в формате "Имя — Роль".

Где работает: Только в группах.

/setrole [user] Роль

+роль [user] Роль

Описание: Назначает роль пользователю в чате.

Как работает: Добавляет запись в Роли Пользователей и Роли. Требуется право на команду.

Пример: /setrole @GiveMyOscar Модератор.

Где работает: Только в группах.

/unsetrole [user]

-роль [user]

Описание: Убирает роль у пользователя.

Как работает: Удаляет запись из Роли Пользователей.

Пример: /unsetrole @GiveMyOscar Модератор.

Где работает: Только в группах.

/mute [time] Причина

Описание: Замьючивает пользователя в ответ на его сообщение.

Как работает: Устанавливает мут в таблице Муты на указанное время (в минутах). В супергруппах ограничивает права, в обычных — удаляет сообщения.

Пример: /mute 30 Спам → мут на 30 минут.

Где работает: Только в группах.

/ban 1d Причина

+бан 1d Причина

Описание: Банит пользователя в ответ на его сообщение.

Как работает: Добавляет запись в таблицу Баны с временем окончания (1d = 1 день).

Пример: /ban 1d Реклама.

Где работает: Только в группах.

/unban [user]

-бан [user]

Описание: Разбанивает пользователя.

Как работает: Удаляет запись из bans.

Пример: /unban @GiveMyOscar.

Где работает: Только в группах.

/kick [user]

Описание: Исключает пользователя из чата.

Как работает: Использует API Telegram для кика, без записи в БД.

Пример: /kick @GiveMyOscar.

Где работает: Только в группах.

/unmute [user]

-мут [user]

Описание: Снимает мут с пользователя.

Как работает: Удаляет запись из Муты или снимает ограничения через API.

Пример: /unmute @GiveMyOscar/p>

Где работает: Только в группах.

/banlist

баны

Описание: Показывает список забаненных пользователей.

Как работает: Запрашивает данные из таблицы Баны.

Где работает: Только в группах.

/mutelist

муты

Описание: Показывает список замьюченных пользователей.

Как работает: Запрашивает данные из таблицы Муты.

Где работает: Только в группах.

/warn Причина

+варн Причина

Описание: Выдаёт предупреждение пользователю (3 предупреждения = бан на 10 дней).

Как работает: Добавляет запись в Варны. При достижении 3 — бан в Баны.

Пример: /warn В ответ на сообщение Флуд.

Где работает: Только в группах.

/unwarn ID

-варн ID

Описание: Снимает одно предупреждение у пользователя.

Как работает: Удаляет последнюю запись из Варны.

Пример: /unwarn В ответ на сообщение.

Где работает: Только в группах.

/warnlist

варны

Описание: Показывает список пользователей с предупреждениями.

Как работает: Запрашивает данные из Варны.

Где работает: Только в группах.

/clear

-чат

Описание: Удаляет последние 100 сообщений пользователя в ответ на его сообщение.

Как работает: Удаляет записи из История сообщений и сообщения через API.

Пример: Ответ на сообщение → /clear.

Где работает: Только в группах.

/pin

Описание: Закрепляет сообщение в чате в ответ на него.

Как работает: Использует API Telegram для закрепления без уведомления.

Пример: Ответ на сообщение → /pin.

Где работает: Только в группах.

/report [причина]

Описание: Отправляет жалобу на сообщение администрации чата.

Как работает: Уведомляет админов и владельца через личку, указывая отправителя, цель, причину и текст сообщения.

Пример: Ответ на сообщение → /report Спам.

Где работает: Только в группах.

Команды владельца (требуют роль "Владелец")

/setwelcome [текст]

Описание: Устанавливает приветственное сообщение для новых участников.

Как работает: Сохраняет текст в таблицу тексты.

Пример: /setwelcome Добро пожаловать в чат!.

Где работает: Только в группах.

/setgoodbye [текст]

Описание: Устанавливает прощальное сообщение для покидающих чат.

Как работает: Сохраняет текст в таблицу тексты.

Пример: /setgoodbye Прощай, друг!.

Где работает: Только в группах.

/zovoffroles

Описание: Включает/отключает упоминание ролей в команде /zov.

Как работает: Меняет значение в Настройки ролей (зовы).

Где работает: Только в группах.

/unsetwelcomegif

Описание: Удаляет приветственный GIF из настроек чата.

Как работает: Устанавливает приветственный GIF в тексты в NULL.

Где работает: Только в группах.

/unsetwelcome

Описание: Удаляет текст приветствия, оставляя GIF (если есть).

Как работает: Убирает приветственное сообщение.

Где работает: Только в группах.

Команды статистики и активности

/stats

+стата

Описание: Показывает статистику пользователя (свою или в ответ на сообщение).

Как работает: Запрашивает данные из Статистика Пользователя (количество сообщений, дата входа и т.д.).

Пример: Ответ на сообщение → /stats.

Где работает: Только в группах.

/mtop

Описание: Показывает топ-10 пользователей по количеству сообщений.

Как работает: Запрашивает данные из Статистика Пользователя, сортирует по message_count, выводит с медалями (🥇🥈🥉).

Где работает: Только в группах.

Команды экономики

/reward

Описание: Выдаёт ежедневный бонус 100 🐾 Lapki Coin (раз в 24 часа).

Как работает: Проверяет время последнего получения, добавляет 100 к balance.

Где работает: Только в группах.

/balance

Описание: Показывает текущий баланс пользователя в 🐾 Lapki Coin.

Как работает: Запрашивает баланс из таблицы.

Где работает: Только в группах.

/topbalance

Описание: Показывает топ-10 пользователей по количеству 🐾 Lapki Coin.

Как работает: Запрашивает данные, сортирует по Балансу, выводит с медалями.

Где работает: Только в группах.

/duel [количество]

Описание: Вызывает на дуэль пользователя (в ответ на сообщение) или всех в чате. Победитель забирает ставку.

Как работает:

  • Минимальная ставка: 150 🐾 Lapki Coin.
  • Создаёт дуэль, ждёт принятия через кнопку в течение 30 секунд.
  • Случайный победитель получает ставку, проигравший теряет.

Пример: /duel 200 (в ответ) или /duel 200 (общий вызов).

Где работает: Только в группах.

/pay [количество]

Описание: Передаёт указанное количество 🐾 Lapki Coin другому пользователю с комиссией 2%.

Как работает: Снимает сумму + комиссию у отправителя, добавляет сумму получателю в coins.

Пример: Ответ на сообщение → /pay 200 → 200 получателю, 4 комиссии.

Где работает: Только в группах.

Игровые команды

/startgame

Описание: Запускает игру "Угадай слово" в чате.

Как работает:

  • Создаёт кнопку "Стать ведущим".
  • Ведущий загадывает слово в личке (до 20 символов).
  • Участники угадывают в чате за 5 минут.
  • Победитель получает 50 🐾 Lapki Coin и становится ведущим, если никто не угадал — ведущий получает 50 🐾.

Где работает: Только в группах.
